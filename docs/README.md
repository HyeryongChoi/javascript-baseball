# 숫자 야구 복습

## 🐾 기능목록

### 📌 UI로직

- [ ] 게임 시작 문구를 출력하는 기능
- [ ] 플레이어가 서로 다른 3개의 숫자를 입력하는 기능
- [ ] 입력한 숫자에 대한 결과를 출력하는 기능

### 📌 도메인로직

- [ ] 컴퓨터가 1~9까지 서로 다른 임의의 수 3개를 선택하는 기능
- [ ] 플레이어가 선택한 숫자를 저장하는 기능
  - [ ] (예외) 1~9로 이루어진 서로 다른 숫자가 아닌 경우
- [ ] 입력한 숫자에 대한 결과를 구하는 기능
  - [ ] 스트라이크: 같은 수가 같은 자리
  - [ ] 볼: 같은 수가 다른 자리
  - [ ] 낫싱: 같은 수가 전혀 없음
